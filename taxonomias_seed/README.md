# taxonomias_seed/

Taxonomías semilla por industria. Sirven como punto de partida para la Etapa 2 del pipeline.

## Estructura

- `salud.yaml` — Reconocimientos Médicos, licencias, turnos, actas, etc.
- `educacion.yaml` — Docentes, escuelas, licencias por artículo, etc.
- `retail.yaml` — Comercios, ventas, presupuestos, stock.
- `servicios_profesionales.yaml` — Abogados, contadores, estudios.

## Convención de nombres

- `taxonomia_<industria>_v<n>.yaml` cuando la versión cambie significativamente.
- Mantener una versión estable por industria hasta que se acumulen muchos cambios.
